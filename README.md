# Python While Loops Exercises

Repositório com exercícios básicos de Python utilizando a estrutura de repetição `while`.

## Conteúdo

Os exercícios abordam conceitos fundamentais como:

- Contagem regressiva
- Soma de números
- Cálculo de média
- Verificação do maior número
- Validação de senha
- Contagem de números pares e ímpares
- Tabuada
- Validação de entrada
- Soma dos dígitos de um número
- Jogo de adivinhação

## Exercícios

### 1. Contagem regressiva até 0
Solicita um número inteiro positivo e exibe a contagem regressiva até 0.

### 2. Soma de números até digitar 0
Recebe números do usuário e soma todos os valores digitados até que 0 seja informado.

### 3. Média dos números digitados
Recebe vários números e encerra quando o usuário digita `-1`, calculando a média dos valores informados.

### 4. Maior número digitado
Solicita números até que 0 seja digitado e exibe o maior valor informado.

### 5. Verificação de senha
Pede ao usuário para digitar a senha correta até acertar.

### 6. Contagem de pares e ímpares
Conta quantos números pares e ímpares foram digitados até o usuário informar 0.

### 7. Tabuada de um número
Exibe a tabuada de 1 a 10 de um número informado pelo usuário.

### 8. Validação de nota
Solicita uma nota entre 0 e 10 e continua pedindo enquanto o valor for inválido.

### 9. Soma dos dígitos
Calcula a soma dos dígitos de um número inteiro positivo usando `while`.

### 10. Jogo de adivinhação
Gera um número aleatório entre 1 e 10 e pede ao usuário para tentar acertar.

## Tecnologias utilizadas

- Python 3

## Objetivo

Este repositório foi criado com foco em prática de lógica de programação e aprendizado da estrutura de repetição `while` em Python.
